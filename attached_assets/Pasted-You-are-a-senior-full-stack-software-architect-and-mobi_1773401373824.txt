You are a senior full-stack software architect and mobile engineer.
Design and build a **modern Restaurant Management System** with a **React Native mobile app** and a **JavaScript backend using Node.js, Express, and MongoDB**.

The system must be **production-ready, scalable, secure, and cleanly structured**.

## 1. TECH STACK

Frontend (Mobile App)

* React Native (Expo or CLI)
* React Navigation
* Axios for API requests
* Redux Toolkit or Zustand for state management
* React Native Paper or NativeBase for UI
* Formik + Yup for form validation

Backend

* Node.js
* Express.js
* MongoDB
* Mongoose ORM
* JWT authentication
* bcrypt for password hashing
* Multer for file uploads
* Cloudinary or similar for image storage

Other tools

* Docker support
* REST API architecture
* MVC folder structure
* Environment variables (.env)
